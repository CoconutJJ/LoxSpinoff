# LoxSpinoff

A language developed following the guide of https://craftinginterpreters.com/the-lox-language.html


## How to use

```git clone https://github.com/CoconutJJ/LoxSpinoff.git```\
```cd LoxSpinoff```\
```make```\
```./interp```


## Expression Syntax
`&&`    AND operator\
`||`    OR operator\
`TRUE`  TRUE\
`FALSE` FALSE\
`+`     PLUS\
`-`     MINUS\
`>`     GREATER\
`<`     LESS\
